# Formation Java et Kotlin

Ce répertoire contient plusieurs projets destinés à l'apprentissage et à la démonstration des concepts de Java, Kotlin, et des microservices avec Spring Boot.

## Projets disponibles

### 1. [Java Intro](./java_intro/README.md)

Un projet d'introduction à Java, couvrant les concepts fondamentaux et avancés à travers des démonstrations pratiques.

### 2. [Kotlin Intro](./kotlin_intro/README.md)

Un projet d'introduction à Kotlin, conçu pour explorer les bases et les fonctionnalités avancées du langage.

### 3. [Microservices](./microservices/README.md)

Un projet démontrant l'architecture des microservices avec Spring Cloud, incluant un serveur Eureka, une passerelle API, et des services de commande et de produit.

### 4. [Spring Boot Demo](./springboot_demo/README.md)

Un projet démontrant l'utilisation de Spring Boot pour créer une application backend avec authentification JWT et gestion des utilisateurs.

## Structure générale

Chaque projet contient :

- **README.md** : Une vue d'ensemble du projet.
- **DOCS.md** : Une documentation détaillée des fonctionnalités et de la structure du projet.
- **Code source** : Situé dans le dossier `src/`.
